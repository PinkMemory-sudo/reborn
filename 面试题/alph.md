# 排序算法

内排序：在内存中完成

外排序：数据量太大，内存中放不下

稳定：排序后相同的元素位置保持不变

不稳定：排序后相同的元素位置会发生变化

就地排序：空间复杂度为O(1)



| 排序算法 | 时间复杂度 | 空间复杂度 | 稳定性 |
| -------- | ---------- | ---------- | ------ |
| 冒泡排序 | n*n        | 1          | 稳定   |
| 快速排序 |            |            | 不稳定 |
|          |            |            |        |



## 交换排序



**冒泡排序**

```Java
int tmp = 0;
for (int i = 0; i < a.length; i++) {
    for (int j = 0; j < a.length - 1 - i; j++) {
        if (a[j] > a[j + 1]) {
            tmp = a[j];
            a[j] = a[j + 1];
            a[j + 1] = tmp;
        }
    }
}
```
