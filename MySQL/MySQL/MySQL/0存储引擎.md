# InnoDB的数据存储结构



磁盘与内存交换的基本单位：页。也就是说，在数据库中，不论读一行，还是读多行，都是将这些行所在的页进行加载。也就是说，数据库管理存储空间的基本单位是页（Page），数据库 I/O 操作的最小单位是页。InnoDB中页的大小默认为 16KB。
